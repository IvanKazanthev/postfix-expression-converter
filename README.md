# Postfix Expression Converter

Программа преобразует арифметические выражения, записанные в постфиксной форме, в последовательность инструкций абстрактной вычислительной машины.

## Используемые инструкции

- `LD A` — загрузить операнд A в регистр.
- `ST A` — сохранить содержимое регистра в переменную A.
- `AD A` — прибавить значение A к содержимому регистра.
- `SB A` — вычесть значение A из содержимого регистра.
- `ML A` — умножить содержимое регистра на значение A.
- `DV A` — разделить содержимое регистра на значение A.

Результат выполнения каждой операции остаётся в регистре.

## Возможности

- обработка постфиксных выражений;
- поддержка операций `+`, `-`, `*`, `/`;
- использование стека на связном списке;
- генерация инструкций вычислительной машины;
- проверка корректности входных данных;
- обработка ошибочных выражений.

## Структура проекта

- `main.py` — пользовательское меню и запуск программы;
- `logic.py` — обработка выражений и генерация инструкций.
